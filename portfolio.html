My current projects

TURNBASED SUPERHERO GAME

A story driven turn based game combining the gameplay from the old school pokemon games and an original hero's journey kind of story that explores subjects like discrimination, overpopulation, pollution, and the overall problems of a modern superpowered society. The player would create his character, be introduced to the world and discover an adventure. The player would then explore the world and choose his story, either saving the world as a super hero, destroying it as a super villain, or something in between. Check out the story so far here.

Virtual Reality Simulation for Law Enforcement Agents in training

Europool suggested this project as a project to do in a class called "Projeto." In this class, my colleague and I chose this project from among other different ones. This project consists of developing a prototype for a simulation for Law Enforcement Agents, experienced or new, in which the user has to conduct a checkpoint operation. This project was developed in Unity, and since it was "commissioned" by Europool, I am not certain that I can link it in this portfolio. CIRCLES OF HELL

A 2d platforms game with a working tutorial for basic movement, attacks, rolls, and a mini level with a Boss-battle that kills the player instantly (it works thematically and there are no more levels.) This project was made in an introduction to game development class, in which they taugth us about GDDs and introduced unity 2d.

MEDIEVAL ADVENTURE GAME

A third person action fantasy game set in a medieval handmade world using a collection of modules from the unity asset store, unity built-in assets and characters from mixamo. The focus of this game is level development. Some notable features being the bridge at the beginning of the level that explodes and forces the player to go through the sewers, the aqueduct that stands over a fog filled valley, the dungeon the player has to cross to get to the castle, and the castle itself.

First game Jam

WORLD TUNER

A third person platformer made in a week-long game jam in which the theme was, in portuguese "Ambiente". "Ambiente" can be interpreted as either ambient or environment. My team (me and another student) decided to read it as ambient and made a game in which the main character can change the ambient music and control the world. Different musics would have different effects in game and could in theory be used to solve multiple puzzles. In the end, we were able to implement 3 of them and built a small map in order to showcase the possible future game. Eventually we plan on finishing it.
